# 🏎️ Engine Shop

**Живая версия → [react-app-engine-shop.vercel.app](https://react-app-engine-shop.vercel.app/)**

Интернет-магазин легендарных JDM-двигателей — пет-проект на **React + Vite**.

Каталог включает культовые моторы Honda, Toyota, Nissan и Subaru: **K20A**, **K24A3**, **3S-GTE**, **1JZ-GTE**, **2JZ-GTE**, **VR38DETT**, **EJ20**.

---

## 🛠️ Технологии

- [React 19](https://react.dev/) — UI-библиотека (классовые и функциональные компоненты)
- [Vite 8](https://vite.dev/) — сборка и dev-сервер
- [React Icons](https://react-icons.github.io/react-icons/) — иконки корзины и удаления
- [ESLint](https://eslint.org/) — линтинг кода
- CSS — стилизация без сторонних UI-библиотек

---

## ✨ Функциональность

- 📦 **Каталог двигателей** — карточки с фото, названием, описанием и ценой
- 🔍 **Фильтрация по категориям** — Honda / Toyota / Subaru / Nissan / Всё
- 🛒 **Корзина покупок**
  - добавление товара по клику на «+»
  - защита от дублей: один и тот же двигатель не добавится дважды
  - удаление позиции из корзины
  - автоматический подсчёт итоговой суммы
  - состояние «Товаров нет» для пустой корзины
- 🖼️ **Детальный просмотр товара** — клик по изображению открывает карточку в увеличенном виде
- 🧭 Шапка с навигацией и презентационным баннером

## 🚀 Запуск проекта

```bash
# Клонировать репозиторий
git clone https://github.com/abylaikhan444/ReactAPP-EngineShop.git
cd ReactAPP-EngineShop

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

После запуска проект будет доступен по адресу `http://localhost:5173`.

### Другие команды

| Команда           | Описание                        |
| ----------------- | ------------------------------- |
| `npm run build`   | Продакшен-сборка в папку `dist` |
| `npm run preview` | Локальный просмотр сборки       |
| `npm run lint`    | Проверка кода линтером          |

## 🧠 Архитектура

Всё состояние приложения хранится в корневом компоненте `App` и передаётся вниз через props — классический **lifting state up**:

| State           | Назначение                              |
| --------------- | --------------------------------------- |
| `items`         | Полный каталог двигателей               |
| `currentItems`  | Товары, отфильтрованные по категории     |
| `orders`        | Содержимое корзины                       |
| `showFullItem`  | Открыт ли детальный просмотр            |
| `fullItem`      | Товар, показанный в детальном просмотре  |

Обработчики (`addToOrder`, `deleteOrder`, `chooseCategory`, `onShowItem`) объявлены в `App` и прокидываются в дочерние компоненты как колбэки. Данные текут вниз, события — вверх.
