# 🏎️ Engine Shop

Интернет-магазин легендарных JDM-двигателей — пет-проект на **React + Vite**.

Каталог включает культовые моторы Honda и Toyota: **K20A**, **K24A3**, **3S-GTE**, **1JZ-GTE**, **2JZ-GTE**.

## 🛠️ Технологии

- [React 19](https://react.dev/) — UI-библиотека (классовые и функциональные компоненты)
- [Vite 8](https://vite.dev/) — сборка и dev-сервер
- [ESLint](https://eslint.org/) — линтинг кода
- CSS — стилизация без сторонних UI-библиотек

## 🚀 Запуск проекта

```bash
# Клонировать репозиторий
git clone https://github.com/<ваш-логин>/ReactAPP-EngineShop.git
cd ReactAPP-EngineShop

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

После запуска проект будет доступен по адресу `http://localhost:5173`.

### Другие команды

| Команда           | Описание                        |
| ----------------- | ------------------------------- |
| `npm run build`   | Продакшен-сборка в папку `dist` |
| `npm run preview` | Локальный просмотр сборки       |
| `npm run lint`    | Проверка кода линтером          |

## 📁 Структура проекта

```
src/
├── assets/            # Изображения (двигатели, баннер)
├── components/
│   ├── Header.jsx     # Шапка сайта с навигацией
│   ├── Items.jsx      # Список товаров
│   ├── Item.jsx       # Карточка товара
│   └── Footer.jsx     # Подвал сайта
├── App.jsx            # Главный компонент, данные каталога
├── main.jsx           # Точка входа
└── index.css          # Глобальные стили
```

## ✨ Функциональность

- 📦 Каталог двигателей с карточками товаров (фото, название, описание, цена)
- 🧭 Шапка с навигацией и презентационным баннером
- 🛒 Кнопка добавления в корзину _(в разработке)_
